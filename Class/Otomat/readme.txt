dòng 1: S -  Tập các trạng thái của otomat
dòng 2: Sigma - bảng chữ cái đầu vào

dòng 5: các hàm chuyển trạng thái (có dạng: ví dụ (trạng thái bắt đầu, trạng thái đến, chữ cái))
